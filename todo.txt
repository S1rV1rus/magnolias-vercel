PROYECTO: SISTEMA DE GESTIÓN PARA CLÍNICA ESTÉTICA "MAGNOLIAS"

DESCRIPCIÓN GENERAL:
Desarrollo de un sistema integral de gestión en la nube diseñado a medida para una clínica estética. El sistema abarca la gestión interna de pacientes (historias clínicas y cuponeras), la administración de los profesionales y sus horarios, una agenda interactiva avanzada (turnos y salas), y facturación básica.
Como diferencial fundamental y fase final, el sistema actuará como "cerebro" para un asistente virtual de WhatsApp (conectado vía ManyChat y n8n) que podrá leer información y agendar citas automáticamente sin intervención humana.

PILA TECNOLÓGICA:
- Frontend (App Web): React (Vite), TypeScript, Tailwind CSS, Shadcn/ui.
- Backend & Base de Datos: Supabase (PostgreSQL, Auth, RLS).
- Automatización & Chatbot: ManyChat, n8n, WhatsApp API.
